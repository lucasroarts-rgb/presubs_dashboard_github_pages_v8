# Audit overview, v10.4

The first dashboard page now follows an account-audit visual structure.

It includes:

- account health score
- 8-dimension radar
- findings by severity
- category score versus target
- 30-day performance pulse
- campaign breakdown
- funnel and creative diagnostics
- waste and opportunity Pareto views
- findings and recommendations
- 30-day action plan

Google Ads-only metrics from the visual reference, such as impression share, search terms, keyword match type and quality score, were not copied because they do not exist in the current Meta Ads dataset. Meta equivalents are used instead.
