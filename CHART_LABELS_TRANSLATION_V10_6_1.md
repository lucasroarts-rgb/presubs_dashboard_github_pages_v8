# v10.6.1 — Chart labels and localization

## Performance Pulse improvements

- horizontal grid lines are now visible
- the left axis shows spend values in EUR
- the right axis shows registrations or rolling CPL values
- the last point of the registrations and CPL series shows its value
- KPI sparklines now include reference grid lines and numeric labels
- tooltips and interactive series controls remain available

## Localization improvements

Expanded Portuguese and French translations for:

- Performance Pulse
- Audit Overview
- Daily Briefing
- Student Profile
- campaign/ad-set/ad views
- date analysis and custom periods
- management and quality sections
- chart legends, empty states and dynamic labels

English remains the default language.

## Installation

Run MIGRAR_DA_VERSAO_ATUAL.bat and select the existing v10.6 folder.
Then run PUBLICAR_NO_GITHUB.bat.
