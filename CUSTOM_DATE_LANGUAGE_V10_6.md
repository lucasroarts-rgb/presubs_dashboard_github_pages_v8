# v10.6 custom dates and languages

- Global selector supports Weekly or Custom dates.
- Custom ranges are built from published daily Meta history, so 5–7 July or any available interval can be selected.
- Previous comparison uses the immediately preceding period with the same number of days.
- Interface language selector: English (default), French and Portuguese.
- Student Profile hides the global campaign-performance hero while all other non-audit views retain it.
- Audit overview includes Period intelligence and weekday performance for meeting context.
