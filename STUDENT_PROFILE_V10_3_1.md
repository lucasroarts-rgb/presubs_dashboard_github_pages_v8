# Student profile, v10.3.1

The dashboard now includes a **Student profile** menu based on aggregated data from:

- 2,001 completed Peasy Academy placement tests
- 114 completed Fluency Club placement tests
- 2,115 completed tests in total

The page includes filters for All students, Peasy Academy and Fluency Club, plus:

- age distribution
- country distribution
- current English level
- learning motivations
- barriers to progress
- discovery channels
- programme features that attract students
- Academy versus Fluency Club comparison
- strategic marketing implications

Only aggregated counts and percentages are included. Emails and individual free-text responses are not included in the package or public dashboard.

The Meta daily automation does not update this section. A future placement-test export must be aggregated into a new dashboard version to refresh the student profile.
