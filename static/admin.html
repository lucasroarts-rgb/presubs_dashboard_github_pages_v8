Dashboard administration

Import weekly and daily results.

Use three weekly exports plus one optional ad export with the Day breakdown. Reimporting the same period updates the existing data.

Meta Ads exports

The first three files must have no breakdown. Only the fourth ad file should use Breakdown → Time → Day.

Ad naming rule

The dashboard identifies the conversion page directly from the ad name.

Examples: [LP-A] ad1630 becomes Page A; [LP-B] ad1629 becomes Page B. Any ad without an [LP-...] tag is assigned to Main page. Duplicate an ad when changing its page so the historical attribution remains correct.

Optional page settings

Register the URL and metadata for a code already used in the ad name. The performance calculation itself is automatic.

Optional external page metrics

Use this only when you later want to add GA4, WordPress or database page views. Meta performance by page is already calculated automatically from ad names.
